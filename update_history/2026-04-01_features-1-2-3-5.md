## What Changed

### Feature #1: Rate Limiting on Yahoo Finance Proxy
- Added per-IP rate limiter to `functions/api/chart/[[path]].js` and `functions/api/options/[[path]].js`
- 30 requests/minute per IP using `CF-Connecting-IP` header (falls back to `x-forwarded-for`)
- Returns 429 with `Retry-After: 60` header when exceeded

### Feature #2: Request Logging / Error Monitoring

- Format: `{ ts, fn, action, status, durationMs, error? }` — captured by Cloudflare dashboard
- Added `window.onerror` and `unhandledrejection` handlers to index.html frontend

### Feature #3: Config Endpoint for Google Client ID
- New edge function `functions/api/config.js` — returns `{ googleClientId }` from env vars
- Updated `index.html`: `loadAppConfig()` fetches `/api/config` on init
- Replaced hardcoded `'GOOGLE_CLIENT_ID'` placeholder with `appConfig.googleClientId`
- Added `/api/config` route to `server.js` for local dev

- `market-pulse.js` now reads `FROM_EMAIL` env var (falls back to `alerts@optionsranker.com`)
